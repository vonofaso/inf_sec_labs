## Комплекс лабораторных работ по дисциплине «Основы информационной безопасности»

**Цель работ:**  
Изучение системного подхода к обеспечению информационной безопасности, включая нормативное регулирование, моделирование угроз и практические методы анализа защищенности.

**Содержание работ:**
- Анализ требований регуляторов в области ИБ и стандартов категорирования объектов критической информационной инфраструктуры (КИИ).
- Построение типовой модели нарушителя и определение векторов атак для различных объектов информатизации.
- Проведение разведки по открытым источникам (OSINT) с применением общедоступного инструментария.
- Изучение основ пентеста: работа с базами данных уязвимостей (CVE, NVD) и первичное сканирование в специализированных дистрибутивах.

**Результат:**  
Сформировано понимание жизненного цикла угрозы — от моделирования нарушителя до поиска уязвимостей. Освоены инструменты OSINT и базового анализа защищенности.
